## Install

### MacOS

```shell script
brew tap psmarcin/tap https://github.com/psmarcin/homebrew-tap
brew install psmarcin/tap/psutils
```

### Go
```shell script
go get github.com/psmarcin/psutils
```

## Usage

```shell script
psutils --help
```
